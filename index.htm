Run It Yourself
# Install (no API keys needed)
pip install requests beautifulsoup4 tqdm

# Scrape all free sources
python analysis/scrape_jobs.py

# Or target specific sources
python analysis/scrape_jobs.py --sources remoteok arbeitnow --limit 500
python analysis/scrape_jobs.py --source hn --hn-thread 43332022

# Aggregate → dashboard-ready JSON (no LLM, rule-based)
python analysis/aggregate_free.py

# Open dashboard
open web/index.html
Methodology

Skills, role types, and seniority are extracted with rule-based regex — zero LLM cost. Salary is normalized to annualized USD. Each scrape writes a timestamped JSON file to data/raw/ and appends to data/raw/scrape_log.jsonl. Trend data is derived from the scraped_at field on every record — no hardcoded dates.
